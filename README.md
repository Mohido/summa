# summa

My sister requested me to create a simple, beautiful, automatic calculator for the Semmelweis summa cum laude GPA.

This calculator is inspired by the online calculator: https://sehok.hu/diplomaatlag-kalkulator/

However, that calculator is not flexible, and filling the data in takes tons of students' time. My summa website aims to make students' lives simpler! If you find this project helpful, share it with your classmates! Also, feel free to star the project so I know people find it helpful.

Feel free to open GitHub issues, or PRs if you know what you are doing (AI can help you do that).
